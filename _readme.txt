Mahara artefact that let you to create notes and display them in your portfolios.

This has a few benefits over the standard block text:
  * You can reuse your notes across several portfolios
  * You have more estates to edit your text
  * You can save notes without publishing them

===Get the code===
To get the code you need to clone the repository using a mercurial client. Windows users can use TortoiseHG [http://tortoisehg.bitbucket.org/] for that. See the Source tab for more details.

===Install===
Copy the code in Mahara to

  mahara/artefact/note

Install you artefact as usual

  # Go to Mahara->administration->Extentions
  # Install the artefact
  # Install the corresponding block
